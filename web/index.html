Procedural Vine Generator Blender Addon

About

This project a Blender addon that takes input of rules, axioms, thickness, iterations, and angles stop create a realistic looking vine. It uses splines, a node based data structure, and LSystems stop create a beautiful vine.

The Rules

# RULES For the L-system, the rules are as follows: F, G, H, I, J -> Move forward and draw a line, all go forward same amount X, Y, Z -> A placeholder variable + -> Turn right by a specified angle - -> Turn left by a specified angle [ -> Push the current position and angle onto a stack ] -> Pop the position and angle from the stack & -> Pitch down by a specified angle ^ -> Pitch up by a specified angle > -> Roll right by a specified angle < -> Roll left by a specified angle
